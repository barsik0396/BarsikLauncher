# Changelog

Все значимые изменения проекта документируются здесь.

## Формат версий

```
версия-канал+blapi-версия_api-git:коммит-api (коммит-версии)
```

Пример: `1.0.0-stable+blapi-v1.0-git:e04ec5f (e04ec5f)`

| Поле | Описание |
|------|----------|
| `версия` | Версия лаунчера |
| `канал` | Канал релиза (`stable`, `beta`, `dev`, ...) |
| `версия_api` | Версия BarsikLauncher API (blapi) |
| `коммит-api` | Коммит blapi, с которым собран релиз |
| `коммит-версии` | Коммит самого лаунчера |
