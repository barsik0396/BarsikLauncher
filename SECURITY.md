## Сообщить об уязвимости

**Не создавай публичный Issue для уязвимостей безопасности.**

Есть два способа сообщить об уязвимости:

### 1. GitHub Private Vulnerability Reporting (рекомендуется)

Используй встроенный инструмент GitHub — кнопка **«Report a vulnerability»** во вкладке **Security** репозитория. Репорт будет виден только мейнтейнерам.

### 2. Email

Напиши на **Barsik0396@outlook.com** с темой `[SECURITY] BarsikLauncher`.

Включи в сообщение:
- Описание уязвимости
- Шаги для воспроизведения
- Возможные последствия
- Предложение по исправлению (если есть)

## Что ожидать

- **Подтверждение** получения репорта — в течение 1–4 дней (иногда дольше)
- После исправления ты будешь упомянут в changelog (если не против)

Спасибо, что помогаешь делать BarsikLauncher безопаснее!
